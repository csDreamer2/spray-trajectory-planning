# 测试模型文件说明

## 📁 目录说明

此目录用于存放STEP格式的3D模型文件，用于测试喷涂轨迹规划系统。

## 🚫 大文件说明

由于GitHub文件大小限制（100MB），原始的大型STEP文件未包含在仓库中：

- `杭汽轮总装.STEP` (622MB) - 完整的汽轮机总装模型
- `MPX3500.STEP` (142MB) - MPX3500型号工件模型

## 📥 获取测试文件

如需完整的测试文件，请：

1. **联系作者**：王睿 (浙江大学)
2. **自行准备**：使用您自己的STEP格式CAD文件
3. **创建简单测试文件**：使用CAD软件导出小型STEP文件

## 🔧 支持的文件格式

- **STEP文件** (.step, .stp) - 主要支持格式
- **IGES文件** (.iges, .igs) - 部分支持
- **STL文件** (.stl) - 网格格式支持

## 📋 文件要求

- **文件大小**：建议小于100MB
- **几何复杂度**：支持复杂曲面和装配体
- **单位**：支持毫米(mm)、米(m)等标准单位
- **编码**：支持ASCII和二进制格式

## 🧪 测试建议

1. **小文件测试**：先使用简单几何体测试基本功能
2. **逐步增加复杂度**：从单个零件到装配体
3. **性能测试**：观察不同文件大小的加载时间
4. **质量模式**：测试快速/平衡/高质量三种加载模式
