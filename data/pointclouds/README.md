# 点云数据文件说明

## 📁 目录说明

此目录用于存放PLY格式的点云数据文件，用于测试点云处理和可视化功能。

由于GitHub文件大小限制（50MB警告），原始的大型PLY文件未包含在仓库中：

- `凝气下半大未分割.ply` (52.58MB)
- `排缸下半未分割.ply` (56.66MB)

- `调阀未分割.ply` (51.27MB)
- `凝汽下半（大）未分割.ply` (55.05MB)
- `大轴承座未分割.ply` (50.70MB)

## 🔧 支持的文件格式

- **PLY文件** (.ply) - 主要支持格式
- **PCD文件** (.pcd) - PCL标准格式
- **XYZ文件** (.xyz) - 简单坐标格式

## 📋 文件要求

- **文件大小**：建议小于50MB
- **点数量**：支持百万级点云数据
- **坐标系**：支持标准笛卡尔坐标系
- **颜色信息**：支持RGB颜色数据（可选）

## 🧪 生成测试点云

可以使用项目中的Python脚本生成简单测试点云：

```python
# 运行测试脚本
python tests/create_simple_step.py
```

## 📊 点云处理功能

系统支持以下点云处理功能：

1. **加载和显示**：支持大型点云文件的高效加载
2. **滤波处理**：噪声去除和数据清理
3. **分割算法**：自动分割和区域识别
4. **特征提取**：关键点和特征识别
5. **可视化**：实时3D渲染和交互
